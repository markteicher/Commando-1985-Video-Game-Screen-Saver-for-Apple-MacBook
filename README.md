# Commando 1985 Video Game Screen Saver for Apple Mac Book
Commando 1985 as a vertically scrolling run and gun shooter developed by CapCom. This is a screen saver for Command 1985 video game players



1. Open CommandoScreenSaver.xcodeproj in Xcode.
2. Select the CommandoScreenSaver target.
3. Under Signing & Capabilities, select your Development Team if Xcode requests it.
4. Product > Build.
5. Find CommandoScreenSaver.saver in Products, right-click > Show in Finder.
6. Double-click CommandoScreenSaver.saver to install it in macOS Screen Saver settings.

This project uses original procedural graphics inspired by the general top-down 1980s arcade-shooter format; it does not contain copied Commando game assets.
